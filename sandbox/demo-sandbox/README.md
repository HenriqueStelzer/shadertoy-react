# Demo sandbox

Multi-tile browser sandbox — same layout as [upstream shadertoy-react Demos](https://codesandbox.io/s/434qm4x4w0) (image fade-in, mouse, clock, orientation, classic GLSL, custom uniforms).

- StackBlitz: https://stackblitz.com/github/HenriqueStelzer/glsl-helpers-react/tree/main/sandbox/demo-sandbox
- CodeSandbox: https://codesandbox.io/s/github/HenriqueStelzer/glsl-helpers-react/tree/main/sandbox/demo-sandbox

For the full 15-tile grid see README [Demo](../../README.md#demo). For a single annotated shader see [Playground](../playground/).

```bash
npm install
npm run dev
```
