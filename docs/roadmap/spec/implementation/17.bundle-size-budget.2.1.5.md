# Bundle size budget

## Goal

Keep lib bundle small; enforce in CI.

## Current state

Baseline ~41 KB min / ~11 KB gzip.

## Deliverables

- [ ] Document baseline in README or `docs/performance.md`
- [ ] CI guardrail (size-limit or bundlesize)
- [ ] Quick wins: dedupe preprocess, lazy optional modules
- [ ] README size note

## Acceptance criteria

1. CI fails PRs over budget
2. Target ≤32 KB min / ≤9 KB gzip documented

## Files and commands

| Path / command | Purpose |
|----------------|---------|
| `lib/glsl-helpers-react.min.js` | Bundle |
| CI workflow | Size check |

## Out of scope

- Example app bundle (~270 KB)
