# 2.1.0

- **Release date:** 2026-06-09 (pending publish)
- **Type:** Minor — demo fixes and `iMouse` coordinate correction

## Summary

**Package:** `glsl-helpers-react@2.1.0`

### Fixes

- **`iMouse` coordinate space** — mouse position now uses backing-store pixels (`CSS × devicePixelRatio`), matching `iResolution` and `fragCoord` on retina displays
- Extracted `toShaderPixelCoords()` helper in `GlslCanvas` for consistent mouse down/move/lerp handling

### Examples

- Demo grid **scroll discoverability** — bottom gradient fade, “scroll for more demos” cue, sticky jump menu with anchor links per tile
