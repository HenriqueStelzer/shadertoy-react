## Summary

**Package:** `glsl-helpers-react@2.1.2`

### Fixes

- **Canvas sizing** — `ResizeObserver` + unified `syncCanvasSize()`; backing store tracks CSS layout × `devicePixelRatio`
- **`iMouse` on scaled canvases** — `toShaderPixelCoords()` uses canvas width/height vs layout rect (not only DPR)
- **Image textures** — defer shader compile until textures load; `flipY` normalized for WebGL (`toGlFlipY`)
- **Multi-pass** — inject `sampler2D iChannelN` for pass `inputs` (fixes undeclared channel uniforms on final pass)
